Project Overview:
This project was developed as part of the Ethara AI assessment process. The application is designed to provide a clean, responsive, and user-friendly experience with modern UI components and smooth navigation.

Technologies Used:
- React.js
- Vite
- Tailwind CSS
- JavaScript
- React Router DOM
- Node.js
- Express.js
- MongoDB
- Clerk Authentication

Key Features:
- Responsive Design
- Modern User Interface
- Secure Authentication
- Smooth Navigation
- Reusable Components
- Mobile Friendly Layout
- Optimized Performance

Project Structure:
- Components
- Pages
- Assets
- Routing
- Styling
- Backend Configuration
- Database Integration

Setup Instructions:

1. Clone the repository

2. Open the project folder

3. Install dependencies:
   npm install

4. Run development server:
   npm run dev

5. Build the project:
   npm run build

Deployment:
The project is deployed using Railway.
